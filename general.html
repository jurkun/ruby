Usage

— If you want to call it by javascipt. Required jquery and app/script.js. 
openLoader([object args, function callback]);
Name Type Description
wrapper string Optional - Name of wrapper element. The default wrapper is body tag.
type string Optional - Loader type (horizontal, circular). The default type is horizontal.
callback function Optional
Usage Example:
openLoader(
{
	wrapper: '#your-wrapper',
},
function (){
	console.log('loader callback');
});
