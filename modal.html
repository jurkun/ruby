Modal

Sample style and usage for modals.

1 Example

— Example modal / popup dialog.

2 Usage

— If you want to call it by javascipt. Required jquery.

openModal(object args[, function callback]);
Name Type Description
wrapper string Required - Identifier of modal element wrapper.
type string Optional - Type default, small, large.
callback function Optional
Usage Example:
openModal(
{
	wrapper: '#your-modal-wrapper'
},
function (){
	console.log('modal callback');
});
