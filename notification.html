2 Usage

— If you want to call it by javascipt. Required jquery.

openNotification(object args[, function callback]);
Name Type Description
wrapper string Optional - Name of wrapper element. The default wrapper is body tag.
style string Optional - Styling notification (default, error, info, success, warning).
type string Optional - Type default or popup.
title string Optional - Your notification title.
message string Required - Your notification message.
callback function Optional
Usage Example:
openNotification(
{
	wrapper: '#your-wrapper',
	style: 'error',
	type: 'popup',
	title: 'Title',
	message: 'Sample message',
},
function (){
	console.log('notification callback');
});
